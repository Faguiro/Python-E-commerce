# Instalação:

```
pip install django
```

```
virtualenv env
```

# Para Mac/ Linux:

```
source env/bin/activate
```

# Para Window:

```
env\scripts\activate
```

```
pip install -r requirements.txt
```

```
python manage.py makemigrations
```

```
python manage.py migrate
```

```
python manage.py runserver
```

# Para logar como Admin:
